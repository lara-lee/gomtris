# Gomtris — LEE Edition

HTML5 + CSS + 순수 JavaScript(Vanilla)로 만든 블록 퍼즐 게임.
외부 라이브러리·빌드 없이 동작하며, Y2K 포스터 감성 디자인 + **보드 밖을 돌아다니는 젤리곰**이 특징입니다.

> 이름은 곰(Gom) + 트릭스의 합성어 **Gomtris(곰트릭스)**. "Tetris" 단어를 피해 상표 위험을 낮췄습니다.

## 실행 방법

```
index.html 을 더블클릭하여 브라우저에서 열기
```

별도 서버·빌드 불필요 (스크립트를 순서대로 로드하는 일반 방식이라 `file://` 에서 바로 실행).

## 조작법

| 동작 | 키보드(PC) | 터치(폰) |
|------|-----------|----------|
| 좌우 이동 | ← / → | 좌우 드래그 |
| 소프트 드롭 | ↓ | 아래 드래그 |
| 회전 / 역회전 | ↑·X / Z | 탭 |
| 하드 드롭 | Space | 아래로 빠르게 튕김 |
| 홀드 | C / Shift | 길게 누르기 |
| 일시정지 | P / ESC | ⏸ 버튼 |
| 재시작 / 음소거 | R / M | — |

> **폰(앱)에는 키보드가 없어도** 됩니다. 보드를 손가락 제스처로 조작하고,
> 화면 하단 터치 버튼 패드(작은 화면에서 자동 표시)로도 모든 조작이 가능합니다.

## 기능

### 게임플레이
- 7가지 표준 블록 + **7-Bag 랜덤**(고른 분포)
- 충돌·줄제거·중력, **줄 제거 깜빡임 애니메이션**
- **고스트 블록**(착지 위치 미리보기), **HOLD / NEXT** 미리보기, 간단한 **월 킥**
- 시작 **카운트다운(3·2·1·GO)**, 일시정지·게임오버·재시작
- 창 포커스를 잃으면 자동 일시정지

### 점수 · 레벨
- 1·2·3·4줄 = 100·300·500·800점 (×레벨)
- **Level(속도)**: 10줄마다 +1, 낙하 속도 증가 (무한)
- HUD: Score / High(localStorage 저장) / Level / Lines / **Gom 단계(1~11)**

### 🐻 젤리곰
- 보드 밖(포스터 헤더·푸터·여백)을 좌우로 **걸어다님**
- **색** = 가장 최근에 줄을 지운 **블록의 색** (랜덤 아님)
- **크기** = 곰 단계 = `floor(지운 줄 / 10) + 1`, **1단계 작게 → 11단계 크게**
- 블록 색 = 곰 색 팔레트(핑크·주황·노랑·초록·파랑)로 통일
- 곰 이미지: `gomimg/<색>.png` (pink·orange·yellow·green·blue, white는 예비 보관)

### 사운드
- **효과음**: 파일 없이 WebAudio 합성음 (이동·회전·드롭·줄제거·레벨업 등)
- **배경음(BGM)**: `assets/audio/bgm.mp3` (선택) — 레벨업 시 템포 상승, M키 음소거

### 디자인
- Y2K 포스터 프레임(각진 검정 외곽선, 그림자 없음), 무지개 그라디언트 제목
- 시작/계속/다시시작 버튼은 중앙 윈도우 오버레이 안에 배치
- 반응형(모바일 터치 패드 자동 노출)

## 디렉토리 구조

```
07.gomtris/
├── index.html              # 진입점 (마크업 + 스크립트 로드)
├── README.md
├── assets/audio/
│   ├── README.md           # 오디오 안내
│   └── bgm.mp3             # (선택) 배경음 — 직접 넣기
├── gomimg/                 # 곰 이미지 (pink/orange/yellow/green/blue/white .png, goms.png 원본)
├── docs/
│   └── tetris-game-plan.html   # 기획안
└── src/
    ├── css/style.css       # Y2K 포스터 테마 + 곰/블록 스타일
    └── js/
        ├── config.js       # 전역 상수 (보드/점수/속도/블록→곰색 매핑)
        ├── audio.js        # 효과음(합성) + 배경음(BGM)
        ├── tetrominoes.js  # 블록 정의 + 7-Bag 랜덤
        ├── board.js        # 게임판 상태 / 충돌 / 줄제거
        ├── renderer.js     # DOM 격자 렌더링
        ├── jellybear.js    # 보드 밖 젤리곰 (색=블록색, 크기=단계)
        ├── game.js         # 게임 루프 / 규칙 총괄
        ├── input.js        # 키보드 입력
        ├── touch.js        # 폰 터치 제스처
        └── main.js         # 부트스트랩 (UI 연결, 시작)
```

설계 원칙: **관심사 분리** — 데이터(board)·표현(renderer)·규칙(game)·입력(input/touch)·소리(audio)·캐릭터(jellybear)를 각 파일로 분리.
