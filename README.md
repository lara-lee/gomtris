# 🐻 Gomtris · 곰트리스

**곰 젤리 + Tetris = 곰트리스.** 젤리곰 색의 블록으로 즐기는 깔끔한 블록 퍼즐 게임.

## 기술 스택

- **HTML5 + CSS3** — 순백 배경 · 픽셀풍 멀티컬러 제목 · 플랫 캔디 블록, 반응형
- **Vanilla JavaScript** — 프레임워크·빌드 도구 없음 (`index.html` 더블클릭으로도 실행)
- **DOM 격자 렌더링** — Canvas 아님. 각 칸이 `div`, 블록은 CSS 클래스로 표현
- **WebAudio API** — 효과음 실시간 합성 (음원 파일 불필요) + 선택적 BGM
- **PWA** — Service Worker(오프라인 캐시) + Web App Manifest(설치형)

## 배포

- 🌐 **PWA**: https://gorgeous-truffle-914e1c.netlify.app/
- 📲 **설치**: 위 주소 접속 → "홈 화면에 추가"(iOS) / "앱 설치"(Android) → 전체화면·오프라인 실행

## 게임 설명

떨어지는 블록을 회전·이동해 가로 줄을 채우면 사라지고 점수를 얻는 클래식 블록 퍼즐. 보드가 화면의 메인이고, 블록 색은 젤리곰 색입니다.

### 조작

| 동작 | 키 (PC) | 모바일 |
|---|---|---|
| 좌우 이동 | ← / → | 좌우 드래그 |
| 빨리 내리기 | ↓ · Space (누르고 있기) | 꾹 누르기 |
| 회전 / 역회전 | ↑ · X / Z | 탭 |
| 일시정지 · 다시시작 · 음소거 | P · R · M | 툴바 버튼 |

### 점수 · 레벨

- 줄 제거: **1줄 100 · 2줄 300 · 3줄 500 · 4줄(테트리스) 800점** (레벨이 높을수록 배수)
- 빨리 내리기 +1점/칸
- **10줄마다 레벨 +1** → 낙하 속도 증가 + 배경음 템포 상승(최대 1.6배)
- 최고 점수는 브라우저(`localStorage`)에 저장

## 핵심 기능

- **7-Bag 랜덤** — 7종을 섞은 묶음 단위로 순차 등장 (편향 방지)
- **월 킥** — 회전 시 막히면 좌우로 살짝 보정 (오프셋 0, ±1, ±2)
- **고스트 블록** — 착지 위치를 점선으로 미리 표시
- **줄 제거 애니메이션** — 깜빡임 후 삭제
- **WebAudio 합성 효과음** — 이동·회전·드롭·줄제거·레벨업·게임오버 등
- **PWA 오프라인 캐시** — Service Worker(네트워크 우선 → 오프라인 시 캐시 폴백)
- **반응형** — 모바일에서 보드를 화면에 꽉 차게(폭 가득 · 높이 맞춤)

## 블록 색 (= 젤리곰 색)

| 블록 | 곰 | HEX |
|---|---|---|
| I · J | 파랑 곰 | `#6ed3f7` |
| O | 노랑 곰 | `#fdd22c` |
| T | 핑크 곰 | `#fd95b2` |
| S | 초록 곰 | `#aed41c` |
| Z · L | 주황 곰 | `#fd6208` |

> 색은 `assets/gomimg/`의 젤리곰 이미지에서 추출했습니다.
