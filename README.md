# Gomtris — Y2K Edition

HTML5 + CSS + 순수 JavaScript(Vanilla)로 만든 블록 퍼즐 게임.
이미지·외부 라이브러리 없이 **CSS만으로** Y2K 포스터 감성(각진 검정 외곽선·네온 블록)을 구현하고,
보드 뒤에서 **젤리곰**이 걸어다니며 줄을 지울수록 성장합니다.

> 이름은 곰(Gom) + 트릭스의 합성어 **Gomtris(곰트릭스)**. "Tetris" 단어를 피해 상표 위험을 낮췄습니다.

## 실행 방법

별도 빌드/서버 없이 동작합니다.

```
index.html 을 더블클릭하여 브라우저에서 열기
```

## 조작법

| 키 | 동작 | 터치(폰) |
|----|------|----------|
| ← / → | 좌우 이동 | 좌우 드래그 |
| ↓ | 소프트 드롭 | 아래 드래그 |
| ↑ / X (Z=역회전) | 회전 | 탭 |
| Space | 하드 드롭 | 아래로 빠르게 튕김 |
| C / Shift | 홀드 | 길게 누르기 |
| P / ESC | 일시정지 | — |
| R | 재시작 · M | 음소거 |

하단 터치 버튼 패드도 보조로 제공됩니다.

## 기능

- 7가지 표준 블록 + **7-Bag 랜덤**
- 충돌·줄제거·중력, **줄 제거 깜빡임 애니메이션**
- **고스트 블록**, **HOLD / NEXT** 미리보기, 간단한 **월 킥**
- 점수 / 레벨 / 라인, 레벨업 시 낙하 속도 증가
- **난이도 3단계**: 초보·중수·고수 (레벨은 항상 1에서 시작, 난이도는 낙하 속도)
- **젤리곰 성장**: 10줄마다 색 젤리가 곰에게 날아와 붙고(색 섞임), 50·80·100줄에 진화 (👑 100줄=황제)
- 시작 **카운트다운(3·2·1·GO)**, 일시정지·게임오버·재시작
- 창 포커스 잃으면 자동 일시정지
- **도움말 패널**, 최고 점수 저장(localStorage)
- **폰 터치 제스처** 지원
- **효과음**: 파일 없이 WebAudio 합성음
- **배경음(BGM)**: `assets/audio/bgm.mp3` (선택) — 레벨업 시 템포 상승

## 디렉토리 구조

```
07.tetris/
├── index.html              # 진입점 (마크업 + 스크립트 로드)
├── README.md
├── assets/
│   └── audio/
│       ├── README.md       # 오디오 안내
│       └── bgm.mp3         # (선택) 배경음 — 직접 넣기
├── docs/
│   └── tetris-game-plan.html   # 기획안
└── src/
    ├── css/
    │   └── style.css       # Y2K 포스터 테마
    └── js/
        ├── config.js       # 전역 상수 (보드/점수/속도/난이도)
        ├── audio.js        # 효과음(합성) + 배경음(BGM)
        ├── tetrominoes.js  # 블록 정의 + 7-Bag 랜덤
        ├── board.js        # 게임판 상태 / 충돌 / 줄제거
        ├── renderer.js     # DOM 격자 렌더링
        ├── jellybear.js    # 보드 뒤 젤리곰 성장 시스템
        ├── game.js         # 게임 루프 / 규칙 총괄
        ├── input.js        # 키보드 입력
        ├── touch.js        # 폰 터치 제스처
        └── main.js         # 부트스트랩 (UI 연결, 시작)
```

설계 원칙: **관심사 분리** — 데이터(board) · 표현(renderer) · 규칙(game) · 입력(input/touch) · 소리(audio)를 각 파일로 분리.
