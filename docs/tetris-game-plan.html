Gomtris(곰트리스) — 기획·구현 명세

곰 젤리 + Tetris = 곰트리스. 젤리곰 색의 블록으로 즐기는 깔끔한 블록 퍼즐. 이 문서는 실제 구현된 코드 기준으로 정리한 명세입니다.

1. 개요

항목내용
게임명Gomtris (곰트리스)
장르퍼즐 / 아케이드 (1인)
플랫폼웹 브라우저 (PC / 모바일 반응형) · 설치형 PWA
기술 스택HTML5 + CSS3 + Vanilla JavaScript (프레임워크·빌드 도구 없음)
렌더링DOM 격자 (Canvas 아님) — 각 칸이 div, CSS로 블록 스타일
사운드WebAudio 실시간 합성 효과음 + 선택적 BGM
실행index.html 더블클릭(file://) 또는 웹 배포 모두 동작
디자인순백 배경 · 픽셀풍 멀티컬러 제목 · 플랫한 캔디 블록 (보드가 메인)

2. 핵심 게임 규칙

2.1 게임판

  • 표준 규격 가로 10칸 × 세로 20칸 (CONFIG.COLS / CONFIG.ROWS)
  • 2차원 배열로 관리, 빈칸은 null / 채워진 칸은 블록 타입 문자

2.2 블록 (Tetromino) — 색은 젤리곰 색

블록모양곰 색HEX
I · J일자 · J자파랑 곰#6ed3f7
O정사각형노랑 곰#fdd22c
TT자핑크 곰#fd95b2
SS자초록 곰#aed41c
Z · LZ자 · L자주황 곰#fd6208
  • 등장 순서는 7-Bag 방식 (7종을 섞은 묶음 단위로 순차 등장)
  • 색은 assets/gomimg/의 젤리곰 이미지에서 추출한 값

2.3 조작

동작키 (PC)모바일
좌/우 이동← / →좌우 드래그
빨리 내리기↓ · Space (누르고 있기)꾹 누르기
회전 / 역회전↑ · X / Z탭
일시정지 · 다시시작 · 음소거P · R · M툴바 버튼
  • 회전 시 막히면 좌우로 살짝 밀어주는 월 킥(오프셋 0, ±1, ±2) 적용
  • 고스트 블록으로 착지 위치를 점선으로 미리 표시

2.4 줄 제거 · 게임 오버

  • 가로 한 줄이 가득 차면 깜빡임 애니메이션 후 삭제, 위 블록이 아래로 내려옴
  • 동시에 여러 줄 제거 가능 (4줄 = 테트리스)
  • 새 블록 생성 자리에 이미 블록이 있으면 게임 종료

3. 점수 · 레벨

동시 제거 줄 수점수 (레벨 n)
1줄 (Single)100 × n
2줄 (Double)300 × n
3줄 (Triple)500 × n
4줄 (Tetris)800 × n
  • 빨리 내리기 +1점/칸
  • 레벨: 10줄마다 +1 → 낙하 속도 증가 + 배경음 템포 상승(최대 1.6배)
  • 최고 점수는 localStorage에 저장(BEST)

4. 화면 구성 (UI)

┌───────────────────────────────┐
│   🐻  G O M T R I S            │   ← 작은 젤리곰 로고 + 픽셀 멀티컬러 제목
│        SCORE 0 · BEST 0        │   ← 점수 한 줄 (텍스트)
│  ┌─────────────────────────┐   │
│  │                         │   │
│  │     게임판 (10 × 20)      │   │   ← 메인. 흰 바탕 + 연한 격자
│  │                         │   │
│  └─────────────────────────┘   │
│   [⏸ 일시정지] [❔ 설명] [🔊]    │   ← 툴바
└───────────────────────────────┘
  • 보드를 메인으로 크게. 상단은 제목 + 점수 한 줄만 (NEXT/HOLD 패널은 두지 않아 보드에 집중)
  • 시작 / 일시정지 / 게임오버는 보드 위 오버레이 창으로 표시
  • 모바일은 보드를 화면에 꽉 차게 — 폭은 가득, 세로가 모자라면 높이에 맞춤(--cell 반응형)

5. 구현 현황 (전부 완료)

  • 게임판 배열 · DOM 격자 렌더링
  • 7종 블록 정의 · 7-Bag 랜덤 · 생성/낙하/이동/회전
  • 충돌 판정 · 줄 완성 감지/제거 · 게임 오버
  • 점수 · 레벨 · 낙하 속도 증가 · 최고 점수 저장
  • 빨리 내리기 · 월 킥 · 고스트 블록
  • WebAudio 합성 효과음 + BGM(레벨업 시 템포 상승)
  • 키보드 + 모바일 터치(좌우 드래그·탭·꾹 누르기) 조작
  • 시작/일시정지/재시작/게임오버 화면 · 설명 패널
  • PWA(Service Worker 오프라인 캐시 + 설치형 manifest) · 반응형
