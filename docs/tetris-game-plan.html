테트리스(Tetris) 게임 기획안

1. 개요

항목내용
게임명(가칭) Classic Tetris
장르퍼즐 / 아케이드
플랫폼웹 브라우저 (PC / 모바일 반응형)
플레이 인원1인
기술 스택(제안)HTML5 Canvas + JavaScript (또는 React + Canvas)
목표떨어지는 블록을 회전·이동시켜 가로 줄을 완성하면 줄이 사라지고 점수를 얻는 클래식 테트리스 구현

2. 핵심 게임 규칙

2.1 게임판 (Board)

  • 표준 규격: 가로 10칸 × 세로 20칸 (그 위 2~4칸은 블록 생성용 버퍼)
  • 좌표는 (행, 열) 기준 2차원 배열로 관리

2.2 블록 (Tetromino)

7가지 표준 블록 사용:

블록모양색상(예시)
I일자 4칸하늘색
O정사각형노란색
TT자보라색
SS자초록색
ZZ자빨간색
JJ자파란색
LL자주황색
  • 블록 등장 순서는 7-Bag 방식(7종을 한 묶음으로 섞어 순차 등장) 권장 → 한쪽 블록만 계속 나오는 불공정 방지

2.3 조작

동작키(PC)모바일
좌/우 이동← / →화면 좌우 버튼 또는 스와이프
소프트 드롭(빨리 내리기)↓아래 스와이프
하드 드롭(즉시 바닥)Space아래 플릭
회전↑ 또는 Z/X탭
일시정지P / ESC일시정지 버튼
  • 회전 시 벽이나 블록에 막히면 위치를 살짝 밀어주는 월 킥(Wall Kick) 적용 권장

2.4 줄 제거 & 중력

  • 가로 한 줄이 빈칸 없이 채워지면 해당 줄 삭제
  • 삭제된 줄 위의 블록들은 아래로 한 칸씩 내려옴
  • 동시에 여러 줄 제거 가능

2.5 게임 오버

  • 새 블록이 생성될 자리에 이미 블록이 쌓여 있으면 게임 종료

3. 점수 시스템

동시 제거 줄 수점수(레벨 n 기준)
1줄 (Single)100 × n
2줄 (Double)300 × n
3줄 (Triple)500 × n
4줄 (Tetris)800 × n
  • 하드 드롭 시 내려간 칸 수만큼 소량 가산점(선택)
  • 레벨 시스템: 일정 줄(예: 10줄)을 제거할 때마다 레벨 +1, 레벨이 오를수록 블록 낙하 속도 증가

4. 화면 구성 (UI)

┌─────────────────────────────┐
│   SCORE: 12300   LEVEL: 3    │
├──────────────────┬──────────┤
│                  │  NEXT     │
│                  │  ┌────┐   │
│    게임판        │  │ 블록 │   │
│   (10 x 20)      │  └────┘   │
│                  │           │
│                  │  HOLD     │
│                  │  ┌────┐   │
│                  │  │     │   │
│                  │  └────┘   │
└──────────────────┴──────────┘
  • NEXT: 다음에 나올 블록 미리보기
  • HOLD: 현재 블록을 보관했다가 나중에 꺼내쓰기 (선택 기능)
  • 고스트 블록: 현재 블록이 떨어질 위치를 반투명하게 표시 (선택)

5. 개발 단계 (로드맵)

1단계 — 핵심 루프 (MVP)

  • 게임판 배열 및 렌더링
  • 7종 블록 정의 및 생성
  • 블록 낙하 / 좌우 이동 / 회전
  • 충돌 판정 (벽, 바닥, 다른 블록)
  • 줄 완성 감지 및 제거
  • 게임 오버 판정

2단계 — 게임성

  • 점수 / 레벨 / 속도 증가
  • NEXT 미리보기
  • 7-Bag 랜덤
  • 하드 드롭 / 소프트 드롭

3단계 — 완성도

  • HOLD 기능
  • 고스트 블록
  • 월 킥(회전 보정)
  • 효과음 / 배경음
  • 최고 점수 저장 (localStorage)
  • 모바일 터치 조작
  • 일시정지 / 재시작 / 시작 화면
